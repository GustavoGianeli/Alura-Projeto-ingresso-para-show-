## Alura-Projeto-ingresso-para-show-

## Uma pagina simples para comprar ingressos para um show  (E-Ticket - Robert Johnson Live)

**Esse é um projeto que eu desenvolvi através dos cursos da Alura** para praticar conceitos de lógica de programação, funções e manipulação de elementos HTML (DOM) utilizando JavaScript. 
  O sistema simula a venda de ingressos, mas foi **customizado por mim para um evento temático** e ganhou novas regras de negócio.

## **O que o projeto faz:**
* Permite selecionar o tipo de ingresso desejado através de um menu de seleção.
* Captura a quantidade informada pelo usuário e valida se há ingressos suficientes no estoque do setor selecionado.
* Subtrai os ingressos vendidos do painel principal e exibe um alerta confirmando o sucesso da compra.

## **Minhas Modificações:**
* **Evento Temático:** Personalizei o sistema focando na venda de ingressos para o lendário show do **Robert Johnson**.
* **Novas Opções e Valores (Camarim VIP):** Expandi o projeto original adicionando o setor de "Camarim VIP". Além de criar a função para gerenciar esse novo ingresso, estruturei o fluxo pensando na lógica de valores específicos e controle rigoroso de quantidades disponíveis para esse setor exclusivo.

## **Tecnologias usadas:**
* HTML5
* CSS3
* JavaScript
